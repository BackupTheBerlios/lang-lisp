Lang(working title) project homepage

This is the home page of the lang project. It is very early in development, you might even find todos in the website. Currently the website is not designed to attract anyone in particular. Please don't advertise this language in a way to suggest that it anywhere near ready to be used for actual programming. Here is the Berlios project page.

What Lang is intended to be

I intend to make a lispy programming language, that treats types better then other lisps. One reason for this is that binaries from this can be faster. In order to produce faster binaries it will be able to produce C code. (I am also considering Forth, because it has a repl.) I will also try to make libraries able to give the user access to many programming methods and concepts. Read further explanation for a more detailed description.(And some considerations.)

License

Because i am 'a follower' of (large portions of)the GNU philosophy, the code of the project will be under the Affero GPL license. It is the Affero license rather then the ordinary GPL to prevent people from providing compilation as a service, and not making improvements of the source code available.

Getting and using the code

The code isn't very user-friendly at the moment, it is available in the git repository, Berlios produces a daily tarball of it. I think much of it is fairly well written. (to-c.lisp and reader.lisp being less good in particular.)

s.lisp contains some 'scrap' code, to playtest, maybe i need to have a more formal usage testings. in test/. It is likely that not everything in s.lisp will work.
